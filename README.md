# WCCA 2020 css styles

## requirements
- nodejs version >= v11.10.1

## clone this repo
```cmd
    git clone git@github.com:elpuas/wcca2020.git
```

## install dependencies
npm
```cmd
    cd wcca2020
    npm install
```
yarn
```cmd
    cd wcca2020
    yarn
```
## run watch mode
npm
```cmd
    cd wcca2020
    npm run dev
```
yarn
```cmd
    cd wcca2020
    yarn run dev
```
## run watch mode with dev server
npm
```cmd
    cd wcca2020
    npm run server
```
yarn
```cmd
    cd wcca2020
    yarn run server
```
## production build
npm
```cmd
    cd wcca2020
    npm run build
```
yarn
```cmd
    cd wcca2020
    yarn run build
```
